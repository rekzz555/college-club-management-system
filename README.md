EVENTUS – College Club Management System

EVENTUS is a web-based College Club Management System developed using PHP, MySQL, HTML, CSS, and JavaScript.
The project helps students register for clubs and events while allowing administrators to manage registrations efficiently.

✨ Features:
Student Login & Registration, 
Admin Login, 
Club Registration System, 
Event Management, 
Feedback System, 
Responsive UI Design, 
Popup Modal Notifications, 
Secure Authentication, 
MySQL Database Integration

🛠️ Technologies Used:
HTML5
CSS3
JavaScript
PHP
MySQL
XAMPP

⚙️ Installation:
Clone the repository
git clone https://github.com/your-username/EVENTUS.git

Move the project folder to:
C:\xampp\htdocs\

Start Apache and MySQL in XAMPP
Create a database named:
eventus

Run the project:
http://localhost/EVENTUS/
